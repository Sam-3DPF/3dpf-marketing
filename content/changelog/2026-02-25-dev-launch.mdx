The new 3D PrintForce platform is live at [dev.3dprintforce.com](https://dev.3dprintforce.com).

This is a full rebuild. The original platform ran on Bubble, which was fine for getting started but started showing its limits as the feature set grew. The new platform is built on FastAPI (backend) and React (frontend) — faster, more reliable, and built to scale without workarounds.

**What's better:** Page loads are significantly faster. The order dashboard handles large order volumes without slowing down. The API layer lets us ship integrations (like Etsy sync) without the constraints we had before.

**Migration from Bubble is in progress.** Existing subscribers are being migrated in batches. You'll get an email when your account is ready on the new platform. In the meantime, your Bubble account stays fully functional.

**If you're new:** Sign up directly at [dev.3dprintforce.com](https://dev.3dprintforce.com). You're starting on the new platform from day one.

More features are already queued up. This launch unblocks a lot of what we've been wanting to ship.
